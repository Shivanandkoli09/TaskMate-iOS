# TaskMate

TaskMate is a UIKit-based iOS To-Do application built using the MVVM architecture.

## Features
- Add, edit, delete tasks
- Persistent storage using UserDefaults
- UIKit + Auto Layout (programmatic UI)
- MVVM architecture
- Task priority and due dates
- Clean and scalable project structure

## Tech Stack
- Swift
- UIKit
- MVVM
- UserDefaults
- Auto Layout
- Git & GitHub
